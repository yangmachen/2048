注意事项：
1.运行前把main执行文件，图片素材下载到和运行路径相同的目录下（建议路径：~/project/）
2.将libmylib.so动态库文件下载到运行路径下的mylib/目录
3.下载后需要修改main文件的权限，执行命令：chmod 777 main
4.运行命令：./main mylib/libmylib.so
